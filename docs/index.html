Installation & Setup

To use FrameCraft with Claude Desktop:

  1. Install via Homebrew: brew install framecraft-mcp
  2. Add to your config:
{
  "mcpServers": {
    "framecraft": {
      "command": "framecraft-mcp",
      "args": []
    }
  }
}

ü§ñ How to use with Claude

Once connected, FrameCraft gives your AI agent the ability to "see" and "draw".

1. Verify Connection

Ask Claude: List available tools. You should see framecraft tools listed.

2. Generate Frames

Simply drag & drop your screenshot into the chat and say:

"Generate an App Store frame for this image. Use the ocean template and the title 'Track your Workouts'."

3. Power User Workflows

  • Batch Generation: "Take these 5 screenshots and generate frames for all of them using the sunset template."
  • Design Exploration: "Show me a list of all available templates first."
  • Custom Text: "Generate a frame with title 'Pro Features' and subtitle 'Unlock the full potential'."
