FrameCraft Automation

Let AI generate frames for you.

1The Easy Way (Recommended)

If you have the Claude Desktop app installed, just paste this into a new conversation:

I want to use FrameCraft automation. Please help me install the 'framecraft-mcp' server using Homebrew. Formula: brew install mario-hernandez/framecraft/framecraft-mcp

Claude will guide you through the rest!

2Manual Installation

If you prefer to use the terminal:

Step 1: Install the server

brew install mario-hernandez/framecraft/framecraft-mcp

Step 2: Configure Claude Desktop

Edit your configuration file located at:

~/Library/Application Support/Claude/claude_desktop_config.json

Add this configuration:

{ "mcpServers": { "framecraft": { "command": "/opt/homebrew/bin/framecraft-mcp", "args": [] } } }
